GitHub PR Notifications

GitHub account

Required for bookmark sync. Create a token with repo scope (classic) or read access to pull requests (fine-grained). Stored locally in your browser and only sent to your GitHub host.

Change this only for GitHub Enterprise (e.g. github.mycorp.com).

Notifications

PR bookmark sync

On every sync, PR bookmarks in the folder are removed and replaced with the current list from the GitHub API. Other bookmarks in the folder are untouched. Status prefix: 🟢 passing · 🟡 pending · 🔴 failing · ⚪ draft · 🟣 merged · ⚫ closed
